A Beginners Guide to Sewing

This website is full of useful links that helped me a lot during my learning process. It seems daunting to learn new things but taking it step by step is key.

What you need to get started

You'll need thread, a sewing machine, scissors, and fabric. This page has useful links to supplies that are needed for sewing. I use the Brother XR9550 sewing machine. I bought it secondhand and it works great. This page has some useful links for sewing supplies.

Helpful tutorials

There are a lot of helpful tutorials you can find online. You can also go to an in-person sewing class if that style of learing is better for you. I am self-taught so it is possible to learn on your own. Bernadette Banner is one of the people that inspired me to get started sewing. She is so talented and has perfect hand stitches and fun sewing videos. Evelyn Wood has so many helpful tutorials that are helpful when learning how to sew. I have many other helpful links on this page.

Free Patterns

Another thing you need to sew cool items is patterns. Patterns are used to make clothing items. The pattern comes on paper, you cut out the shapes, then stitch them together. You can get them for free online or you can purchase them. My first sewing projects were made using free sewing patterns. This is helpful for understanding sewing patterns.
